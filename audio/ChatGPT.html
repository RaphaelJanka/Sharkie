Ihr aktueller Code prüft, ob der Eingabetext am Anfang des Titels oder der Beschreibung eines Tasks steht und blendet den Task aus, wenn dies nicht der Fall ist. Um sicherzustellen, dass die Aufgaben angezeigt werden, deren Titel oder Beschreibung den Buchstaben im Eingabetext enthalten, können Sie den Code wie folgt anpassen:

javascript
function filterTasksByTitle(input) {
    for (let i = 0; i < tasks.length; i++) {
        const taskTitle = tasks[i]['title'].toLowerCase();
        const taskDescription = tasks[i]['description'].toLowerCase();
        const id = tasks[i].id;
        const taskContainer = document.getElementById(`task-${id}`);
        if (!taskTitle.includes(input.toLowerCase()) && !taskDescription.includes(input.toLowerCase())) {
            hideTask(taskContainer);
        } else {
            showTask(taskContainer);
        }
    }
}

In diesem geänderten Code verwenden wir die includes-Methode, um zu überprüfen, ob der Eingabetext in sowohl dem Titel als auch der Beschreibung des Tasks enthalten ist. Dadurch werden alle Tasks angezeigt, deren Titel oder Beschreibung den Buchstaben im Eingabetext enthalten. Beachten Sie, dass wir den Eingabetext und die Titel/Beschreibung in Kleinbuchstaben konvertieren, um sicherzustellen, dass die Suche nicht case-sensitive ist.
